## How Can I Contribute?

### Reporting Bugs

If you find a bug in any of feniqs_lite's components, please open an issue [here](https://github.com/qperfect-io/feniqs_lite/issues) and include:

- A clear and descriptive title.
- A detailed description of the problem.
- Steps to reproduce the issue.
- Any error messages, screenshots or logs.


### Suggesting Enhancements

Enhancement suggestions are welcome! Please open an issue [here](https://github.com/qperfect-io/feniqs_lite/issues) and include:

- A clear and descriptive title.
- A detailed description of the proposed enhancement.
- Why you believe this enhancement would be useful.

 
### Adding New Benchmark Cases

1. Clone the repository (`git clone https://github.com/qperfect-io/feniqs_lite/`).
2. Create a new branch (`git checkout -b feat-new-bench-case-branch`).
3. Create `data` directory (if does not exist yet) and copy the QASM files to it
4. Commit your changes (`git commit -m 'feat: Add new benchmark case'`).
5. Push to the branch (`git push origin feat-new-bench-case-branch`).
6. Open a pull request to the `main` branch of the original repository.

Your pull request should include:
- A clear description of the benchmark case.

Thank you for contributing to feniqs_lite!
